# **App Name**: VelocityType

## Core Features:

- Typing Test UI: Clean and intuitive user interface for taking typing tests.
- Random Text Generation: Fetch and display a random meaningful paragraph using a generative AI tool.
- Performance Metrics: Display typing speed (WPM) and accuracy in real-time.
- Theme Toggle: Toggle between light and dark mode for comfortable use in different environments.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey a sense of focus and sophistication.
- Background color: Very light gray (#F0F0F0) for light mode, dark gray (#333333) for dark mode.
- Accent color: Gold (#FFD700) to highlight important information and interactive elements, giving a touch of prestige.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and short amounts of body text, 'Inter' (sans-serif) for longer body text.
- Minimalist, line-based icons for settings, theme toggle, and other interactive elements.
- Clean, centered layout with a focus on readability and minimal distractions.
- Subtle animations for theme transitions and user interactions, providing a smooth and engaging experience.
